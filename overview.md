11# GTS - Dự án Website Bất Động Sản

## Tổng quan dự án

Đây là website của công ty GTS chuyên về bất động sản, du lịch và các dịch vụ liên quan.

## Cấu trúc dự án

- `app/` - Thư mục chính chứa các trang của ứng dụng Next.js
  - `real-estate/` - Trang bất động sản
  - `tourism/` - Trang du lịch
  - `home/` - Trang chủ
  - `news/` - Trang tin tức
  - `about/` - Trang giới thiệu
  - `contact/` - Trang liên hệ

## Cách chạy dự án

```bash
npm install
npm run dev
```
